If you get HTML responses instead of JSON:
  • Make sure the notification server is running: node notification-server.js
  • Check if port 4000 is available and not blocked
  • Verify the API base URL is correct (should be http://localhost:4000)
  • Check the server console for any error messages
If you get connection errors:
  • Start the notification server: start_notification_server.bat
  • Or manually: node notification-server.js
  • Make sure all dependencies are installed: npm install
